# Resource Dependency Tracker (Strategic Minerals) [under-development]

## What's it About?
A full-stack geopolitical analytics tool designed to track global supply chains for critical minerals (e.g., Lithium, Cobalt, Semiconductors). The application calculates the economic vulnerability of countries based on their resource deficits and visualizes global dependencies to allow countries to identify & plan to DIVERSIFY THEIR IMPORTS.

* **frontend:** ReactJS, HTML, CSS
* **backend:** Python, REST API
